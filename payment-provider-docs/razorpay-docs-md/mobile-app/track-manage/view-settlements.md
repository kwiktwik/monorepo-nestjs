You can use the Razorpay Payments Mobile app to view settlement details.

## How to View Settlements

Follow these steps to view settlements:

1. Log in with your Dashboard credentials.
2. Navigate to **Transactions** → **Settlements**.
3. Tap on a settlement to view more details. You can also filter settlements by:
   - Date

     ![Filter by Date](https://razorpay.com/docs/payments/mobile-app/track-manage/build/browser/assets/images/payments-mobile-app-date-settlements.jpg)
   - Status

     ![Filter by Status](https://razorpay.com/docs/payments/mobile-app/track-manage/build/browser/assets/images/payments-mobile-app-status-settlements.jpg)
