You can cancel an unpaid invoice which is in `issued` state. A customer cannot make payments for the cancelled invoices. Know more about [invoice states](/razorpay-docs-md/invoices/states.md).

## Cancel an Invoice From Dashboard

To cancel an invoice:

1. Log in to the Dashboard.
2. Click **Invoices**.
3. Search for the invoice that you want to cancel using the [search criteria](/razorpay-docs-md/invoices/search.md)   . You can cancel invoices that are in `draft` or `issued` state.
4. Select the **Invoice Id**.
5. On the right-hand side panel, click the **Cancel Invoice** button.
6. On the **Cancel Invoice?** dialog box, click **Yes, Cancel** to confirm the cancellation.

![cancel invoice dialog box](https://razorpay.com/docs/payments/invoices/build/browser/assets/images/invoices-invoice-cancel.jpg)

The cancelled invoices display a **Cancelled** status label.
