**Before you proceed:**

- Understand the [payment flow](/razorpay-docs-md/payment-gateway/how-it-works.md)

- Generate the [API Keys](/razorpay-docs-md/api/authentication.md#generate-api-keys)

  from the Dashboard. To go live with the integration and start accepting real payments, generate Live Mode API Keys and replace them in the integration.

## Supported Versions

We support the following versions of OpenCart with separate extensions: OpenCart 4, OpenCart 3, OpenCart 2 and OpenCart 1.5.
