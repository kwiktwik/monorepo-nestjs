You can close a QR Code. A customer cannot make payments for the closed QR Codes. Know more about [QR Code States](/razorpay-docs-md/qr-codes/states.md).

**Handy Tips**

QR Codes that support multiple payments cannot be closed.

## Close a QR Code from the Dashboard

Here is short video on how to close a QR code from the Dashboard.

To close a QR Code:

1. Log in to the Dashboard.
2. Click on **QR Code**.
3. [Search for the QR Code](/razorpay-docs-md/qr-codes/search.md)

   that you want to close using the search criteria.
4. Select the **QR Code ID**.
5. On the right-hand side panel, click the **Close** button.

   ![Close a QR code from the Razorpay Dashboard](https://razorpay.com/docs/payments/qr-codes/build/browser/assets/images/qr-codes-close-close.jpg)
6. On the **Close QR Code?** dialog box, click **Yes** to confirm the closure.

   ![Confirm the closure of a QR code from the Razorpay Dashboard](https://razorpay.com/docs/payments/qr-codes/build/browser/assets/images/qr-codes-close-confirm-close.jpg)

The closed QR Codes display a **Closed** status label.

![QR code closed from the Razorpay Dashboard](https://razorpay.com/docs/payments/qr-codes/build/browser/assets/images/qr-codes-close-closed.jpg)
