## Installation

You can install Razorpay using Maven or Gradle.

Maven

Gradle

1. Download and install [Maven](https://maven.apache.org/download.cgi)

   on your system.
2. Download the [latest Source code](https://github.com/razorpay/razorpay-java/releases)

   zip file from the Releases section of GitHub.
3. Unzip the file and add this dependency to the project object model (POM) of your project.
