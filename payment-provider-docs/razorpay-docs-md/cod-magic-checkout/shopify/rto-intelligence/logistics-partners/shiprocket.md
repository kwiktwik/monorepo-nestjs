Shiprocket is India's leading logistics software, offering automated shipping solutions.

## Integration Steps

1. Configure Settings on Shiprocket Dashboard

**Handy Tips**

This step is required only if you are generating the API credentials for the first time. If you have already generated the API credentials, please skip this step.

1. Log in to the [Shiprocket Dashboard](https://app.shiprocket.in/login)

   and navigate to **Setup and Settings** → **API** on the bottom right.
2. Click **Configure**.

   ![Navigate to Configure section on the Shiprocket Dashboard.](https://razorpay.com/docs/payments/cod-magic-checkout/shopify/rto-intelligence/logistics-partners/build/browser/assets/images/magic-checkout-shiprocket-login.jpg)
3. In the **API User** section, click **Create an API User**.
4. Enter your **Email ID** and **password**.
5. Click **Generate API Credential**.

   ![Enter the credentials and generate the API keys](https://razorpay.com/docs/payments/cod-magic-checkout/shopify/rto-intelligence/logistics-partners/build/browser/assets/images/magic-checkout-shiprocket-api.jpg)

2. Enable Shiprocket on Razorpay Dashboard

1. Log in to the Dashboard.
2. Navigate to **Magic Checkout** → **Setup and Settings**.
3. Navigate to **RTO Settings** → **Delivery Tracking** and click **Connect** next to **Shiprocket**.
4. Enter your **API User Email ID** and **API User Password** generated on the Shiprocket Dashboard.
5. Click **Connect to Shiprocket**.

   ![Enter your API keys](https://razorpay.com/docs/payments/cod-magic-checkout/shopify/rto-intelligence/logistics-partners/build/browser/assets/images/magic-checkout-shiprocket-credentials.jpg)

You have successfully integrated COD & Magic Checkout with Shiprocket. In case you want to disconnect Shiprocket, click **Disconnect**.
