Razorpay Payment Links let you accept payments without a website or app.

Generate a unique link from your [Dashboard](/razorpay-docs-md/payment-links/create.md) or via [APIs](/razorpay-docs-md/api/payments/payment-links/create-standard.md) and share it with customers via email, SMS, or social media. Customers can use these links to pay securely and easily.

You can manage all Payment Links directly from the Dashboard - create, cancel, duplicate, search, or update them. You can also get instant notifications by subscribing to Webhook events. Most of these actions can also be performed using APIs.

## Advantages

- **Create-to-Collect in a matter of minutes**: Create Payment Links easily with a few clicks and share them with customers.
- **Automatic sharing of link**: The customers receive a Payment Link on SMS or email using which they can quickly pay.
- **Set expiry date**: Set due dates for Payment Links after which the links expire.
- **Partial payments**: Allow your customers to make [partial payments](/razorpay-docs-md/payment-links/partial-payments.md)  . You can also decide the first installment amount to be paid by customers in case of partial payments.
- **Set reminders**: Send automated SMS to your customers to remind them about outstanding payments.
- **Easy integration using APIs**: Seamlessly integrate Payment Links with your systems using APIs.

## List of Supported Payment Methods

**Online Payment**

Customers can make online payments using [Card](/razorpay-docs-md/payment-methods/cards.md), [UPI](/razorpay-docs-md/payment-methods/upi.md), [NetBanking](/razorpay-docs-md/payment-methods/netbanking.md), [Wallet](/razorpay-docs-md/payment-methods/wallets/amazon-pay.md), [EMI](/razorpay-docs-md/payment-methods/emi/debit-card-emi.md), [Pay Later](/razorpay-docs-md/payment-methods/pay-later.md) and [Bank Transfer](/razorpay-docs-md/payment-links/bank-transfer.md) options.

## International Currency Support

You can create Payment Links in any of the supported [international currencies](/razorpay-docs-md/international-payments.md#supported-currencies) using the Dashboard or APIs.

### Address Verification System

If you are accepting international payments, you can use Razorpay's Address Verification System (AVS). AVS verifies if a customer's billing address (postal code and the billing street address) matches the billing address on file with the card issuer. Based on the response from the issuer, Razorpay will accept or cancel the transaction. This helps in the prevention of fraud in international payments. Know more about [Address Verification System](/razorpay-docs-md/international-payments/address-verification-system.md).

## Get Started

Ensure you have a [Razorpay account](/razorpay-docs-md/set-up.md). Log in to the Dashboard and click Payment Links.

## Integrate With Existing Systems

You can easily integrate Razorpay Payment Links with your existing order management and billing solution using the Dashboard. With API integration, a Payment Link can be created as soon as an order is created on your order management systems.
