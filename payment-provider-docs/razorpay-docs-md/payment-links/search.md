You can search for Payment Links on the Dashboard by specifying various filters. Know more about [Payment Link states](/razorpay-docs-md/payment-links/states.md).

## Search a Payment Link From Dashboard

To search a Payment Link:

1. Log in to the Dashboard.
2. Go to the **PAYMENT PRODUCTS** section and click **Payment Links**.
3. In the list of Payment Links that appears, search for the link by specifying filters.

![Search for a payment link](https://razorpay.com/docs/payments/payment-links/build/browser/assets/images/payment-links-search-pl.jpg)
