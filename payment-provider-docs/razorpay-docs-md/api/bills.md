# Bills APIs

You can use Razorpay [Billme](/docs/pos/billme/) to create digital bills and share them with your customers.

Fork the Razorpay Postman Public Workspace and try the Bills APIs using your [Test API Keys](/razorpay-docs-md/dashboard/account-settings/api-keys.md).
