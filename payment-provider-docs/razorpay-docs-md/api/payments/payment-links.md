# Payment Links APIs

Copy for AI

View as Markdown [Payment Links](/razorpay-docs-md/payment-links.md) are URLs that you can send to your customers through SMS and email to collect payments from them. Customers can click on the URL, which opens the payment request page, and complete the payment using any of the available payment methods. You can create, fetch, edit or cancel Payment Links using APIs or from the [Dashboard](/razorpay-docs-md/refunds.md#dashboard-actions).

Fork the Razorpay Postman Public Workspace and try the Payment Links APIs using your [Test API Keys](/razorpay-docs-md/api/authentication.md#generate-api-keys).
