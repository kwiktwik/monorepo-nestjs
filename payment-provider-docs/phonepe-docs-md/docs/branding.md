# Branding

The PhonePe corporate logo is the most immediate representation of the company. It is a must and should be used consistently in the proper, approved forms.

You can download the official PhonePe Logo and images for Integration from **[here](https://drive.google.com/drive/u/1/folders/1hRhOS7w8nd9EZ5XqajMrbZIx0JbCqbbV)**.

You can also refer to Brand Guidelines below:

- Ensure that PhonePe is written as “**PhonePe**” with capital P’s and no spaces.
- The color scheme should not be changed under any circumstances without notification.
