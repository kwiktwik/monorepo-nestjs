# Manage Users

The ГўВҖВң**Manage Users**ГўВҖВқ page empowers administrators to efficiently manage all users associated with their PhonePe Business account. This includes adding new users, assigning roles, defining store-level access, and monitoring user statuses, ensuring secure and appropriate access to the dashboardГўВҖВҷs functionalities.

The PhonePe Dashboard has different types of roles with access to different functionalities.

## LEVEL 1: ADMIN

The Admin can access all sections of Dashboard ГўВҖВ“ Transactions, Profile and Mapped QR Codes as well as trigger refunds on the Transaction Details Page. The Admin can see mapped QR codes and modify notification receivers for them.

Only the Admin can add new users to PhonePe Dashboard with the following roles.

Only the Admin can add new users to PhonePe Insights with following roles.

## LEVEL 2: FINANCE MANAGER

The Finance Manager can access settlement-related and transaction-related pages. He/She is authorized to refund an existing transaction and can also download settlement/transaction reports.

He/She has permissions similar to the Admin, except, they cannot add/modify other user permissions or manage QR/POS devices.

## LEVEL 3: MANAGER

The Manager can access the following sections of Dashboard ГўВҖВ“ Transactions, Profile and Mapped QR Codes as well as trigger refunds on the Transaction Details Page.

The Manager can see mapped QR codes and also modify notification receivers for them. However, a Manager cannot add/modify user permissions.

## LEVEL 4: SUPERVISOR

The Supervisor can access following sections of the Dashboard ГўВҖВ“ Transactions, Profile and Mapped QR Codes. The Supervisor cannot initiate a refund on the Transaction Detail page. S/he can view mapped QR Codes as well as modify notification receivers for them.

The Supervisor does not have access to settlement-related data. An Supervisor also cannot add/edit user roles and permissions.

## LEVEL 5: CASHIER

The Cashier can access the following sections of the Dashboard ГўВҖВ“ Transactions, Profile and Mapped QR Codes. S/he cannot initiate a refund on the Transaction Detail page. The Cashier can see mapped QR Codes. The Cashier also does not have access to settlement-related data, and cannot add/edit user roles and permissions.

1. Log into the PhonePe Dashboard
2. ClickГӮВ **Manage Users**

4. Enter name, contact number and email ID (eg ГўВҖВ“ГӮВ [[emailВ protected]](/cdn-cgi/l/email-protection#2058595a60474d41494c0e434f4d)) of the new user
5. Select one of the following four roles for a new user ГўВҖВ“ Finance Manager, Manager, Supervisor or Cashier
6. ClickГӮВ **Add User**
7. ClickГӮВ **Yes**ГӮВ to confirm

Now, the new user can access PhonePe Dashboard in the assigned role.

## Managing/Editing User Roles

Clicking on a userГўВҖВҷs **Role** in the main user list opens the ГўВҖВңEdit User DetailsГўВҖВқ modal, allowing you to change their assigned role and understand the permissions associated with each role.

#### **How to Edit a UserГўВҖВҷs Role:**

1. On the ГўВҖВңManage UsersГўВҖВқ page, click on the **Role** of the user you wish to modify.
2. In the ГўВҖВңEdit User DetailsГўВҖВқ modal, use the **User Role** dropdown to select the desired new role.
3. Review the description of the selected role to confirm it grants the appropriate permissions.
4. Click ГўВҖВңSaveГўВҖВқ (implied button) to apply the change.

## Managing Store Access

Clicking on the ГўВҖВңManage Store AccessГўВҖВқ link for a user opens a modal that allows you to control which specific stores a user can access and manage data for. This is crucial for multi-store businesses to restrict user access to relevant locations.

#### **How to Manage Store Access for a User:**

1. On the ГўВҖВңManage UsersГўВҖВқ page, click on the **Manage Store Access** link for the desired user.
2. In the ГўВҖВңManage store accessГўВҖВқ modal:
   - If the user currently has access to all stores and you wish to restrict them, toggle **ГўВҖВңRestrict store accessГўВҖВқ** to ON.
   - Use the search bar to find specific stores, if needed.
   - **Check the boxes next to the stores** that this user should be able to view or manage. Unchecked stores will be inaccessible.
3. Click **OK** to save the changes.
