# Introduction to PhonePe Payment Link

---

PhonePe provides a simple and effective Payment Links solution that allows you to collect payments from your customers without any website or app integration. By generating a secure URL that can be shared via SMS, email, or any messaging platform, you can start accepting payments instantly from anywhere.

Whether you’re looking to enable quick one-time payments or need a scalable way to automate bulk link generation, PhonePe Payment Links offer a seamless and reliable experience with minimal setup.

### **STEP 1**

**Create a Link**
Generate secure payment links using the PhonePe Dashboard or API to collect payments for your products or services.

### **STEP 2**

****Share the Link****Send the link to your customers via SMS, email, or any messaging platform of your choice.

### **STEP 3**

**Get Paid Instantly**
Customers can pay using UPI, cards, or net banking, and you can track all payments and settlements in real time from the dashboard.

## Ways to Create and Use Payment Links

### 1. Using PhonePe Dashboard

You can create and manage payment links directly from the PhonePe Dashboard. Here’s how:

**Step 1:** Go to the **Payment Links** tab on your Dashboard.

**Step 2:** Enter payment details such as amount, customer contact information, and link expiry (optional).

**Step 3:** Once the link is created, PhonePe will automatically send it to your customer via SMS or email.

**Step 4:** You can view the status of all your created links in real time. If a payment attempt fails, the customer can retry on the same link. However, only one successful payment is allowed per link.

**Step 5:** Use the **Share** option to resend the link if needed, or cancel it anytime to stop accepting payments.

**Step 6:** All transactions made through payment links will appear under the **Transactions** and **Settlement Reports** tabs.

No coding required. Perfect if you’re looking for a quick, no-integration solution.

### 2. Using API

If you want to automate payment link generation, you can integrate with the Payment Link API.

ℹ️ ****Prerequisite for Using Payment Link API****!

---

To use the Payment Link API, prior approval from the PhonePe team is required. Please contact the PhonePe Business Team to request access. They will guide you through the approval process and enable the necessary configuration for your account.

Once approved, follow these steps:

**Step 1:** Integrate with the Payment Link API by referring to the API documentation.

**Step 2:** Send a request with payment details such as amount, reference ID, customer contact information, and expiry time.

**Step 3:** You will receive the generated link in the API response.

**Step 4:** Share the link with your customer through SMS or Email.

**Step 5:** Use Webhook or Payment Link Status API to track the payment status in real time.
