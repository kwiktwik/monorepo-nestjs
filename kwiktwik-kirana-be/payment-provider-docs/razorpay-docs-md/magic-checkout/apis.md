You can use the following APIs related to Magic Checkout:

- **Coupon Code**: Use the [Coupon Code APIs](/razorpay-docs-md/magic-checkout/apis/coupon-code.md)

  to send coupon details to Razorpay. Razorpay will create the coupon codes, which you can then apply to transactions.
- **Customer Address**: Use the [Customer Address APIs](/razorpay-docs-md/magic-checkout/apis/customer-address.md)

  to upload customers' billing and shipping addresses in bulk.
- **Shipping Information**: Use the [Shipping Information APIs](/razorpay-docs-md/magic-checkout/apis/shipping-info.md)

  to send the customer's shipping information to Razorpay.
