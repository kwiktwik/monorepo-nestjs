You can create a unique Account Code for each linked account, while adding accounts, from the Dashboard.

## Add and Manage Linked Accounts

From the Dashboard, create a linked account and provide the bank account details:

1. Click **Accounts** tab, and then click **+ Add Account**.

   ![Add Account](https://razorpay.com/docs/payments/route/account-code/build/browser/assets/images/route-add-account1.jpg)
2. Enter **Account Name**, **Account Email** and **Account Code**.
3. If you want to enable the Dashboard access for the linked account, turn on the toggle bar.
4. If you want to allow the customer refunds for the linked account, turn on the toggle bar.

   **Handy Tips**

   You must add Email ID to enable Dashboard access and customer refunds for the linked account.
5. Click **Add**.

   ![Add Account pop-up](https://razorpay.com/docs/payments/route/account-code/build/browser/assets/images/route-add-account-pop-up1.jpg)

In the **KYC Form**, enter **Business Details** and **Bank Account Details**, and then click **Submit Form**.

![Submit KYC details](https://razorpay.com/docs/payments/route/account-code/build/browser/assets/images/route-add-account-kyc.jpg)

## Dashboard Access

Once logged in, linked accounts can invite other team members to manage their account.

Also, they can perform other actions such as processing refunds. Know more [about Linked Account Dashboard](/razorpay-docs-md/route/linked-account.md).

**Handy Tips**

- While you can create a linked account without adding their email, you cannot grant them Dashboard access. The access can be granted or provided only after you add the email address of the linked account.

  ![add email to enable Razorpay Dashboard access](https://razorpay.com/docs/payments/route/account-code/build/browser/assets/images/route-add-email1.jpg)
- You cannot access the linked account Dashboard unless they add you as a team member from their linked account Dashboard. [Learn more about Dashboard operations](/razorpay-docs-md/route/batch-upload.md).
