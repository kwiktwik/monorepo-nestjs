We support the following payment methods for instant refunds. The list of banks that support instant refunds differ according to payment methods - [Netbanking](/razorpay-docs-md/refunds/supported-payment-methods.md#netbanking) and [UPI](/razorpay-docs-md/refunds/supported-payment-methods.md#upi).

Refunds will be sent back to the original payment method used in making the payment. For example, if netbanking was used to make the payment, the refund will be pushed to the same netbanking details.
