You can send the Standard Payment Links as URLs to customers through SMS and emails. Customers can open these URLs and make payments using netbanking, cards, wallets, UPI and bank transfer payment methods.

You can also use [UPI Payment Links](/razorpay-docs-md/payment-links/upi.md) to receive UPI payments.

## Payment Link States

Know about the different [Standard Payment Link states](/razorpay-docs-md/payment-links/states.md#standard-payment-links).

## Create

Create a Standard Payment Link from the [Dashboard](/razorpay-docs-md/payment-links/create.md#create-a-standard-payment-link-from-dashboard) or using our [APIs](/razorpay-docs-md/api/payments/payment-links/create-standard.md).
