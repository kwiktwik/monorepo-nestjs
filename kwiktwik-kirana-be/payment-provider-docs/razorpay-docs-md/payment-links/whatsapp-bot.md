Install the Razorpay WhatsApp Bot for Payment Links and accept payments from your customers by creating and sharing Payment Links to them directly from WhatsApp.

## Installation and Configuration

**Handy Tips**

If you lose all your contacts or misplace your mobile phone, you can recover access to the WhatsApp bot by following this process:
Send a text message saying `Create 100` to this phone number: `+91 6364900106`. Please ensure you send this message from your number registered on WhatsApp.

1. Log in to the Dashboard and navigate to the **App Store**.
2. Select the **Payment Links Bot**.

   ![Select Payment Links Bot](https://razorpay.com/docs/payments/payment-links/build/browser/assets/images/pl-app-store-whatsapp-bot.jpg)
3. Click **Install**.

   ![Install Payment Links Bot](https://razorpay.com/docs/payments/payment-links/build/browser/assets/images/pl-whatsapp-bot-install.jpg)
4. You will receive a welcome message on the mobile number which is associated with your Razorpay Admin account.

   ![Install Payment Links Bot](https://razorpay.com/docs/payments/payment-links/build/browser/assets/images/pl-whatsapp-welcome.jpg)
5. To create a Payment Link, Send `Create <Amount>`. Eg. Send `Create 100` to create a Payment Link of INR 100.

   ![Install Payment Links Bot](https://razorpay.com/docs/payments/payment-links/build/browser/assets/images/pl-whatsapp-create.jpg)
6. You will receive the Payment Link from Razorpay which can now be shared with your customers.

   ![Install Payment Links Bot](https://razorpay.com/docs/payments/payment-links/build/browser/assets/images/pl-whatsapp-result.jpg)
