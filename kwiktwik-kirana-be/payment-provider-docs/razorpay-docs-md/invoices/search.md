You can search for invoices (except those marked **Deleted**) on the Dashboard by specifying various filters. Know more about [invoice states](/razorpay-docs-md/invoices/states.md).

## Search an Invoice From Dashboard

1. Log in to the Dashboard.
2. Select **Invoices** from the left menu and search for an invoice under **Invoices** by specifying filters.

![Invoice List](https://razorpay.com/docs/payments/invoices/build/browser/assets/images/invoices-invoices-list-view.jpg)

## Search an Invoice Using API

- View the list of all the invoices or the invoices raised for a customer using the [Fetch Multiple Invoices API](/razorpay-docs-md/api/payments/invoices.md#fetch-multiple-invoices)  .
- View the details of an invoice by providing the invoice id using the [Fetch an Invoice API](/razorpay-docs-md/api/payments/invoices.md#fetch-an-invoice)  .
