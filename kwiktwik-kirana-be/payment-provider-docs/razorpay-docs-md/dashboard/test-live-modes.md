**Watch Out!**

The Dashboard offers two modes: **Test** and **Live**. Both modes provide the same functionalities, except that real payments cannot be accepted in **Test** mode. Refer the [table below](/razorpay-docs-md/dashboard/test-live-modes.md#test-mode-vs-live-mode) for more details.

## Test Mode Vs Live Mode

**Separate Set of API Keys in Test and Live Mode**

You have to generate a separate set of API keys for Live and Test modes. Know more about [Generating API Keys on Dashboard](/razorpay-docs-md/dashboard/account-settings/api-keys.md).

## Enable Test Mode

You can turn on **Test Mode** using the toggle on your Razorpay Dashboard.
