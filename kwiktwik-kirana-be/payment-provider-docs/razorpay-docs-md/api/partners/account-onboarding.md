# Account APIs

You can use the Account APIs to create a sub-merchant account. After an account gets created, an `account_id` is assigned. Check the [account activation flow and states](/docs/partners/aggregators/onboarding-api/#account-activation).

Fork the Razorpay Postman Public Workspace and try the Account APIs using your [Test API Keys](/razorpay-docs-md/api/authentication.md#generate-api-keys). [![Run in Postman](https://run.pstmn.io/button.svg)](https://www.postman.com/razorpaydev/workspace/razorpay-public-workspace/folder/12492020-51a4958d-ae00-4717-9ec4-64cb7c1bf7fb)
