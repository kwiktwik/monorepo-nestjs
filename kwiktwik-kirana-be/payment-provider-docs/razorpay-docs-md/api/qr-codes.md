# QR Codes APIs

Copy for AI

View as Markdown

You can use Razorpay [QR Codes](/razorpay-docs-md/qr-codes.md) to create QR Codes and share them with customers to accept digital payments. You can create and manage QR Codes using APIs or from the [Dashboard](/razorpay-docs-md/qr-codes/create.md).

Refer to the [QR Code Flow](/razorpay-docs-md/qr-codes/how-it-works.md) and [Integration Checklist](/razorpay-docs-md/qr-codes/apis.md#integration-checklist) before you begin integrating with the APIs for QR Codes.

Fork the Razorpay Postman Public Workspace and try the QR Codes APIs using your [Test API Keys](/razorpay-docs-md/dashboard/account-settings/api-keys.md#generate-api-keys). [![Run in Postman](https://run.pstmn.io/button.svg)](https://www.postman.com/razorpaydev/workspace/f960539b-c56c-41e5-810b-63d2a89d447e/folder/12492020-90436eef-f17f-4bc1-9d13-b622a4270125)

Related Guides: [About QR Codes](/razorpay-docs-md/qr-codes.md)
