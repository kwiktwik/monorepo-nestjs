# Customers APIs

Copy for AI

View as Markdown

Add or create [Customers](/razorpay-docs-md/customers.md) with basic details such as name, email and contact details. You can then offer various Razorpay solutions to your customers. Edit customer details as needed. You can create and manage customers using APIs or from the [Dashboard](/razorpay-docs-md/customers.md).

Fork the Razorpay Postman Public Workspace and try the Customers APIs using your [Test API Keys](/razorpay-docs-md/api/authentication.md#generate-api-keys).
