# Setup Subscription using API

---

This API is used to set up a subscription for a customer.

## Environment

|  |  |  |
| --- | --- | --- |
| **Environment** | **HTTP Method** | **API** |
| Sandbox | `POST` | https://api-preprod.phonepe.com/apis/pg-sandbox/checkout/v2/pay |
| Production | `POST` | https://api.phonepe.com/apis/pg/checkout/v2/pay |

## Request

|  |  |  |
| --- | --- | --- |
| **Header Name** | **Header Value** | **Description** |
| Content-Type | application/json |  |
| Authorization | O-Bearer <access\_token> | Pass access\_token received in Authorization call |

## Response

Sample Response

```
{
    "orderId": "OMO123456789",
    "state": "PENDING",
    "expireAt": 1703756259307,     
    "redirectUrl": "../transact/pgv2?token=<token>" 
}
```

Response Parameters

|  |  |  |
| --- | --- | --- |
| **Field Name** | **Data Type** | **Description** |
| `orderId` | String | Internal order ID generated by PhonePe Payment Gateway. |
| `state` | String | State of the order created, expected value is PENDING. |
| `expiryAt` | Long | Order expiry timestamp in epoch format. |
| `redirectUrl` | String | Intent url to invoke the payment app that initiates the mandate and completes the payment. |
